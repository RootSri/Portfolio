Projects Overview

My projects focus on creativity and real-world application while taking a hands-on approach to solving cybersecurity problems. The projects range from Web Bug Hunter, a focused vulnerability scanner made to find web application vulnerabilities like XSS, SQLi, and CSRF, to Intrusion Detection System, a Python-based solution made to monitor, detect, and log suspicious network activities. These projects demonstrate my technical proficiency, critical thinking, and capacity to provide significant, security-focused solutions, whether they are protecting networks through intelligent detection or improving web application security through automation..

Key Areas of Focus

My projects cover a wide range of domains, including vulnerability assessment, penetration testing, secure software development, and network security analysis. Through these hands-on experiences, I have refined my ability to identify and mitigate risks, design resilient systems, and implement proactive security measures. Whether developing custom security tools, automating tasks, or performing forensic investigations, each project reflects my deep understanding of cybersecurity principles and commitment to securing digital systems.

Notable Projects

  • Intrusion Detection System – A Python-based network monitoring tool built to detect suspicious activities and unauthorized access attempts. It demonstrates my ability to implement rule-based threat detection and system-level logging for real-time cybersecurity defense.
  • Web Bug Hunter – A custom vulnerability scanner developed in Python to identify web application flaws such as SQL Injection, XSS, CSRF, and more. This project highlights my skills in offensive security, HTTP analysis, and automation of web penetration testing.

Each project reflects my expertise in cybersecurity, offensive security, and threat detection — showcasing a practical, hands-on approach to solving real-world security challenges.

They reflect my passion for cybersecurity and commitment to making a tangible impact in the field.
